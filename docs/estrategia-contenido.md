# Revisión de estructura, posicionamiento y conversión

## Diagnóstico aplicado

La nueva arquitectura evita presentar a J R Consulting como una lista extensa de servicios. La propuesta principal se articula como una firma que convierte estrategia, tecnología y conocimiento en sistemas implementados. Esto permite incluir líneas distintas sin diluir el posicionamiento.

## Arquitectura propuesta

| Ruta | Función comercial | Llamada principal |
| --- | --- | --- |
| `/` | Explicar la propuesta integral y dirigir cada intención | Hablemos de tu proyecto |
| `/soluciones` | Consultoría, procesos, tecnología e IA | Solicitar diagnóstico |
| `/websites` | Vender websites como herramienta de negocio | Cotizar mi website |
| `/aprende` | Presentar rutas prácticas y capacitación para equipos | Explorar rutas |
| `/research` | Captar protocolos y asesoría metodológica | Revisar mi protocolo |
| `/ramx` | Integrar RAMX como producto del ecosistema | Conocer RAMX |
| `/seguros` | Cotización acompañada de seguros | Cotizar ahora |
| `/contacto` | Separar citas tecnológicas y de negocio | Programar una cita |
| `/cotizar` | Captar necesidades con contexto suficiente | Enviar solicitud |
| `/admin` | Editar contenido y revisar prospectos sin código | Guardar cambios |

## Decisiones de texto

- El inicio vende el resultado antes que la herramienta.
- “Websites” se posiciona con claridad, intención, medición, desempeño y autonomía.
- J R Aprende habla de preparación y rutas hacia certificaciones reconocidas, pero no atribuye alianzas con Google, Coursera u otros proveedores que no estén formalizadas.
- J R Research deja explícitos los límites de autoría, aprobación institucional y responsabilidad sobre los datos.
- RAMX se presenta con las funciones disponibles y diferencia la red veterinaria como siguiente etapa.
- J R OS aparece como “Próximamente”, sin formulario de compra ni promesas de disponibilidad.

## Sistema de llamadas a la acción

- CTA principal de intención alta: cotizar, solicitar diagnóstico o revisar protocolo.
- CTA secundaria de exploración: conocer método, rutas o proceso.
- Agenda de Google específica para tecnología.
- Agenda de Google específica para necesidades generales del negocio.
- WhatsApp visible como salida rápida, con el número `221 375 9147`.
- Los formularios guardan prospectos en el panel y ofrecen WhatsApp como respaldo si la base de datos no está disponible.

## SEO técnico implementado

- Metadatos únicos y canonical por ruta.
- Sitemap y robots.
- Datos estructurados `ProfessionalService`.
- HTML semántico, un H1 por página y jerarquía clara.
- Contenido localizado para Puebla y México sin limitar la atención nacional.
- Core Web Vitals favorecidos mediante tipografías del sistema, assets locales y componentes ligeros.
- Páginas legales y advertencias listas para revisión jurídica.
